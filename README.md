# Video Conferencing App

A full-stack real-time video conferencing application built using the MERN stack, WebRTC, and Socket.IO. The platform enables users to create and join meeting rooms, communicate through video and audio calls, share screens, and exchange messages in real time.

## Features

* User Authentication (Login & Registration)
* Real-Time Video Calling using WebRTC
* Real-Time Audio Communication
* Screen Sharing
* In-Meeting Chat System
* Meeting History
* Multiple Participants Support
* Responsive User Interface
* Secure Room-Based Communication

## Tech Stack

### Frontend

### Backend

* Node.js

### Real-Time Communication

* WebRTC
* STUN Servers

## Installation

### Clone Repository

```bash
git clone https://github.com/chaitanya-kanase/Video-Conferencing-App.git
cd Video-Conferencing-App
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
